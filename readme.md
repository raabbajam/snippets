## Snippets

Snippets is a demo of basic code sharing web apps.

## Installing to local

```bash
git clone https://github.com/raabbajam/snippets &&
cd snippets &&
composer install &&
bower install &&
npm install &&
gulp &&
chmod -R 777 app/storage
```
